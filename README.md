# Terminal ChatBot

A terminal-based AI chatbot built in Python that communicates with Google's Gemini API. The project allows users to interact with an AI assistant directly from the command line without requiring a web interface.

This project was built to learn API integration, Python programming, JSON handling, and terminal application development.

install a venv(virtual environment) to use it.
follow up commands for help are listed below.
* python3 -m venv VirtualEnv
* source VirtualEnv/bin/activate
* pip install google-genai



## Features

* Interactive terminal chat interface
* AI-powered responses using Gemini API
* Continuous conversation loop
* Simple and lightweight design
* Easy to run from the command line
* Beginner-friendly code structure

## Technologies Used

* Python
* Google Gemini API
* JSON
* Virtual Environments (venv)
* Linux / WSL
* Terminal

## How It Works

1. The user enters a message in the terminal.
2. The chatbot sends the message to the Gemini API.
3. The API generates a response.
4. The response is displayed back in the terminal.
5. The conversation continues until the user exits.
